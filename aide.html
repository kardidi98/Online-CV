Conteneur de la page :

1.Interaction facile avec la page.

2.Liste des différentes sections composant votre CV.

3.Possibilité d'ajout et de suppression de formulaires.

Interaction facile avec la page (1) :

Une fois la page est chargée, la barre ci-dessous vous permettra de la manipuler, par exemple : revenir à l'accueil, s'inscrire si vous visitez le site pour la première fois et aussi de se déconnecter.

Interaction facile avec la page (2) :

Le site contient aussi une barre latérale contenant plusieurs options (Accueil, Services, A propos, Contact, Se déconnecter), ce qui vous permez de naviguer dans le site (vous l'obtenez quand vous appuiez sur le bouton qui apparait dans l'image précedente près de nom du site 'GNRCV').

Interaction facile avec la page (3) :

Appuiez sur le bouton 'Aide' au cas où vous voulez savoir plus sur le site et sur les autres boutons qui apparait dans l'image pour enregistrer et télécharger votre CV final.

Liste des différentes sections composant votre CV :

Pour qu'un CV soit établi correctement, il doit d'abord contenir les sections présentées ci-contre(Informations personnelles,Expériences professionnelles,Formation,Outils informatiques,Langues maîtrisées,Centres d'intérêt et Réferences), et par la suite vous aurez la possibilité d'ajouter d'autres sections selon votre choix, ainsi que changer leur ordre en les glissant vers l'endroit voulu.

Possibilité de l'ajout et de la suppression de formulaires :

Les sections 'Expériences professionnelles', 'Formation', 'Outils informatiques' et 'Langues maîtrisée' sont spéciales car elles dépendent du niveau de la personne, c'est pour cela vous avez la possibilité d'ajouter autant de formulaires que vous voulez via le bouton 'ajouter' ou les supprimer un par un via le bouton 'supprimer'.
